Web Technologies Cheat Sheet

World Wide Web (WWW): The web is a system of interlinked hypertext documents accessed via the internet. It was invented by Tim Berners-Lee in 1989.
Key Technologies:

  • HTML: Hypertext Markup Language, used to create web pages.
  • URL: Uniform Resource Locator, the address used to access resources on the web.
  • HTTP: Hypertext Transfer Protocol, used for communication between web servers and browsers.

URL (Uniform Resource Locator): A URL specifies the address of a web resource and is composed of different parts:

  • Protocol: Specifies how the resource is accessed (e.g., HTTP, HTTPS).
  • Domain: The web address of the server (e.g., example.com).
  • Path: The location of a specific page or file (e.g., /about).
  • Query Parameters: Optional data passed to the server (e.g., ?id=123).
Example: https://www.example.com/about?user=123.

Web Server: A web server hosts websites and delivers web pages to clients (web browsers). It processes requests from browsers (e.g., when a URL is entered) and responds with the appropriate resources (e.g., HTML, CSS, images). Examples include Apache and Nginx.

Web Browser: A web browser is a software application that retrieves, displays, and allows users to interact with web pages. Browsers interpret HTML, CSS, and JavaScript to render pages visually. Common browsers include Chrome, Firefox, and Safari.

HTML Elements

<!DOCTYPE>: This declaration defines the document type and HTML version being used. It must be the very first line in an HTML document.
Example: <!DOCTYPE html> (HTML5).

<html>: The root element of an HTML document. It contains all other elements.
Example: <html lang="en"> (The lang attribute specifies the language).

<head>: The head element contains meta-information about the document, such as its title and character encoding. This content is not displayed on the page.
Example: 

<meta charset="UTF-8">
(Specifies the character encoding).

<title>: Defines the title of the document that appears in the browser tab.
Example: <title>Web Technologies Cheat Sheet</title>.

<body>: Contains the visible content of the web page, such as text, images, and links.
Example: <body>This is the main content of the page</body>.

<strong>: Marks text as important and typically renders it in bold.
Example: <strong>Important!</strong>.

<em>: Marks emphasized text, which is typically rendered in italics.
Example: <em>Emphasized text</em>.
