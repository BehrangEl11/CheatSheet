Web Technologies Cheat Sheet

World Wide Web (WWW)

The World Wide Web (WWW) is a system of interlinked hypertext documents that are accessed through the internet. It was created by Tim Berners-Lee in 1989. The three essential technologies behind the web are: - **HTML** (Hypertext Markup Language): The standard language for creating web pages. - **URL** (Uniform Resource Locator): The address used to locate web resources. - **HTTP** (Hypertext Transfer Protocol): The protocol for transferring web pages between clients and servers.

URL (Uniform Resource Locator)

A URL is an address that points to a resource on the web. The structure of a URL includes: - **Protocol**: Defines the method to access the resource (e.g., HTTP, HTTPS). - **Domain**: Identifies the website (e.g., example.com). - **Path**: Specifies the location of the page or file on the server (e.g., /about). - **Query**: Optional parameters passed to the server (e.g., ?id=123). Example: `https://www.example.com/about?user=123`

Web Server

A web server is responsible for storing, processing, and delivering web pages to clients (web browsers). When a browser sends a request to a web server, it responds with the requested resource (e.g., an HTML page). Common web servers include Apache and Nginx.

Web Browser

A web browser is software that retrieves, displays, and allows users to interact with web pages. It sends requests to web servers and renders the returned content, such as HTML, CSS, and JavaScript. Popular browsers include Chrome, Firefox, and Safari.
